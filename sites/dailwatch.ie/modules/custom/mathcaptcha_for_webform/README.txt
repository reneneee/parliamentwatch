Mathcaptcha_for_webform module offering general CAPTCHA
administration and a simple math challenge.

Dependencies
------------
1. captcha module
2. captcha_pack module

Installation:
-------------
  Installation is like with all normal drupal modules:
  extract the 'Mathcaptcha_for_webform' folder from the tar ball to the
  modules directory from your website (typically sites/all/modules/custom/).

Configuration:
--------------
  Goto admin->config->people->captcha
  Paste webform_client_form_17 in form_id and select "Math CAPTCHA(from module math_captcha)" from Challenge type dropdown
